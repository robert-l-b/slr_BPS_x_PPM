## 🔍 Overview

As discussed in the paper, predictive process monitoring (PPM) models are primarily used to parameterize or refine business process simulation (BPS) models. In particular, these models:

- Predict the future behavior of running process instances  
- Support the generation of event data  
- Capture different process perspectives (e.g., control-flow, temporal, resource, and data)
